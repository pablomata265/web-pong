# Pong Game

## Description
The Pong game is a classic arcade game that simulates table tennis. Players control paddles to hit a ball back and forth, aiming to score points by getting the ball past the opponent's paddle.

## Features
- Two-player mode
- Simple and intuitive controls
- Scoring system
- Adjustable difficulty levels

## How to Play
1. Start the game from the main menu.
2. Each player controls a paddle. Player 1 uses the **W** and **S** keys to move up and down, while Player 2 uses the **Up Arrow** and **Down Arrow** keys.
3. The objective is to score by getting the ball past your opponent's paddle.
4. The game ends when a player reaches the required score.

## Controls
- **Player 1:**
  - W: Move up
  - S: Move down

- **Player 2:**
  - Up Arrow: Move up
  - Down Arrow: Move down

Enjoy the game!
